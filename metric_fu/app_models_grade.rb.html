1# Grade represents a grade given by the course instructor.
  
2#
  
3# Grade allows the professor to grade students. Professor can go to Grade by clicking Grade tab in faculty
  
4# tools, on the index page of each course. Professor can assign/view/change the score of the student directly
  
5# by putting the score for the assignment in front of his/her name. Also, only one grade is permitted for one student
  
6# per assignment. The design of grade is to encapusulate the grading from the submission.
  
7#
  
8# * For a course, a student will have at most one grade on each assignment.
  
 9# * is_student_visible indicates that whether this grade is going to publish to student or not.
  
10# * score would be in two forms. If the grading rule is set to use points, the score should be a number greater than
  
11#   zero, and we don't validate whether the score is greater than maximum number defined in Assignment object, so that
  
12#   professor can add extra credit on student's grade. If the grading rule is set to use letter grades, score would be
  
13#   A, A-, B+, B, B-, C+, C, or C-.
  
14# * score= assigns a number greater than zero, and we don't validate whether the score is greater than maximum
  
15#   number defined in Assignment object, so that professor can add extra credit on student's grade.

1# GradingRule represents the correspondence between points earned and letter grades. The mapping rule is given by the
  
2# course instructor.
  
3#
  
4# GradingRule is configurable by clicking on "Configure course" in "Initial Course Configuration", both of which could
  
5# be found on the index page of each course. In the configuration page, professor can choose grading criteria between
  
6#   points and letter grades. If professor adopts letter grades, then professor needs to fill the table which maps out
  
7#   point and letter grades.
  
8#
  
 9# GradingRule follows the grading policy of CMU@SV. We provide the following options for letter grades: A, A-, B+, B,
  
10#   B-, C+, C, C-. These grades can be configured by faculty by going to course configuration page. Beside these grades
  
11#   faculty can also enter R, W, I grades. But these grades are not configurable.
  
12#
  
13# We provide the following functions to map out points and letter grades.
  
14# * validate_letter_grade performs letter grade validation for the given score.
  
15# * validate_score performs score validation for the given score by the grade type
  
16# * format_score formats the given score
  
17# * get_grade_in_prof_format convert points to
  
18# * grade_type tells whether the deliverable is graded by weight or by points
  
19# * A_grade_min tells on or above this value the student will be marked as A-
  
20# * A_minus_grade_min tells on or above this value the student will be marked as A
  
21# * B_plus_grade_min tells on or above this value the student will be marked as B+
  
22# * B_grade_min tells on or above this value the student will be marked as B
  
23# * B_minus_grade_min tells on or above this value the student will be marked as B-
  
24# * C_plus_grade_min tells on or above this value the student will be marked as C+
  
25# * C_grade_min tells on or above this value the student will be marked as C
  
26# * C_minus_grade_min tells on or above this value the student will be marked as C
  
27# * course_id tells the course for which this grading rule exists
  
28# * is_nomenclature_deliverable tells which name is preferred by professor (deliverable or assignment)

57  def validate_score(raw_score)
  
58    # allow users to skip entering grades
  
59    if raw_score.nil? || raw_score.empty?
  
60      return true
  
61    end
  
62
  
63    # allow users to enter letter grades
  
64    if mapping_rule.has_key?(raw_score.to_s.upcase)
  
65      return true
  
66    end
  
67
  
68    # return the grading type
  
69    case grade_type
  
70      when "points"
  
71        return validate_points(raw_score)
  
72      when "weights"
  
73        return validate_weights(raw_score)
  
74      else
  
75        return false
  
76    end
  
77  end

125    ["A_grade_min", "A_minus_grade_min", "B_plus_grade_min", "B_grade_min", "B_minus_grade_min", "C_plus_grade_min", "C_grade_min", "C_minus_grade_min"].each do |attr_name|
